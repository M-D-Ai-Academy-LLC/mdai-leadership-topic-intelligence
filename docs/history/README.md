# History Index

This directory centralizes historical records for repository activity.

## Files

- `docs/history/repository-history.md` - Commit-level timeline with key changes.
- `docs/history/pull-request-history.md` - Merged PR timeline and merge commits.
- `docs/history/workspace-state-log.md` - Local workspace state snapshots.

## Update Guidance

- Update `repository-history.md` when significant commits are added.
- Update `pull-request-history.md` after PR merges to `main`.
- Update `workspace-state-log.md` before and after large local cleanup operations.
